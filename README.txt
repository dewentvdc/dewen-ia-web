# Ombligo Net SAS

Este paquete incluye:
- Landing Page
- Portal de Clientes
- Panel de Administración
- Backend Node.js + SQLite
- Botón de Pago Wompi

## Instalación
1. Ir a backend/: `npm install && node server.js`
2. Ir a frontend/: `npm install && npm start`
3. Subir a Render o Railway para hosting gratuito.
